Administrator's Database Panel

It's a simple administrator's system built with Java, Spring Boot (with Spring Security) and PostgreSQL. I've created it for someone who needed help in organizing his work. The system has 2 roles: ADMIN and USER. You can login using [justUser, Passing123] or [justAdmin, Passing123] as username and password. If the user is not admin, some routes are restricted. I am going to extend this project with additional functions. Click live to see it on Heroku; Source button takes you to the source code on GitHub.
